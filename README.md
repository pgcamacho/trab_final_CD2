Projeto final para o curso Ciência de Dados Aplicada à Epidemiologia II
Os dados de mortalidade foram extraídos do banco previamente organizado para o projeto de tese de doutorado em epidemiologia.
O projeto tem como tema o câncer em adolescentes e adultos jovens de 15 a 39 anos, na população de treze Registros de Câncer de Base Populacional (RCBP) brasileiros, no período de 2001 a 2017.
